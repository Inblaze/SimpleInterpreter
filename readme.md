### 函数绘图语言解释器

源码位于目录`./MyInterpreter`中，在装有GNU的C++编译器的Windows下运行该目录下的`compile.bat`文件即可生成解释器的可执行程序`MyInterpreter.exe`

该目录中的`1.txt`为语义分析器的测试用例。
